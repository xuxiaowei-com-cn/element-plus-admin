# element-plus-admin

## 创建项目

~~~
npm install -g @vue/cli
npm init @vitejs/app my-vue-app --template vue
~~~

## 添加路由

~~~
npm i vue-router@next
~~~

## 添加缓存

~~~
npm i vuex@next
~~~
